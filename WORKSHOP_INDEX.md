# AWS First Cloud Journey Workshop Index

This index collects the public hands-on AWS workshop repositories in one place so learners can navigate by topic instead of searching across many numbered repositories.

Total indexed workshops: **136**

## Recommended Learning Path

1. Start with core AWS account, IAM, VPC, EC2, and S3 labs.
2. Move into serverless, containers, and managed databases.
3. Practice security, observability, cost controls, and CI/CD.
4. Use the advanced labs for migration, high availability, and production patterns.
